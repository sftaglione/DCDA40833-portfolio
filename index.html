Semester Reflection

What I learned: In this lab, I learned how to set up a GitHub repository, edit files using Visual Studio Code, and deploy a live website using GitHub Pages. I also gained a better understanding of how HTML and CSS work together to structure and style a portfolio.

Challenges I overcame: One challenge I faced was troubleshooting issues with file organization and CSS styling, especially when adjusting image size and layout. I worked through this by carefully reviewing the instructions, testing changes step by step, and using feedback to improve my site.

How I grew: This lab helped me become more comfortable working with unfamiliar tools and showed me how digital projects are built iteratively rather than all at once.

Looking ahead: As the semester continues, I plan to expand this portfolio with data visualizations, interactive projects, and deeper reflections on digital culture and analytics.
