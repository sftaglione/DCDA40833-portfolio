Hometown Mapping

Sophia Taglione | DCDA 40833 | Spring 2026

Lab Requirements

For this lab we created an interactive hometown map using Python. I built a dataset of locations from my hometown of Dana Point, organized the information in a CSV file, and used Python mapping tools to generate an interactive map with markers and pop-up descriptions for each location.

What This Lab Demonstrates

This lab demonstrates my ability to combine coding, data organization, and visualization. By using Python to generate an interactive map, I was able to turn location data into something visual and engaging. It also shows how coding can be used to present real places and experiences in a digital format.

Interactive Map

Below is the interactive hometown map I generated using Python and Folium. Each marker represents a location in Dana Point that I selected for the project, including restaurants, beaches, and scenic viewpoints. When users click on a marker, a pop-up appears with a short description and image of the location.

Creating this map helped me better understand how coding can turn raw data into something interactive and meaningful. Instead of just listing locations in a dataset, I was able to visualize them in a way that shows patterns and context, like how many of my favorite places are centered around the harbor area. It also showed me how small coding decisions, like customizing markers or pop-ups, can improve how users experience and interact with the data. I also learned that troubleshooting small issues, like file paths and map rendering, is just as important as building the map itself.

Reflection

One of the biggest challenges I faced in this lab was figuring out how to properly connect my Python-generated map to my HTML portfolio page. At first, I kept running into issues where the map would overwrite my entire page or not display at all. This forced me to slow down and really understand how files are connected, especially the difference between generating a file in Python and embedding it into an existing webpage.

Working through these problems helped me become more comfortable troubleshooting and thinking through code step by step. Instead of just copying solutions, I had to test different approaches and figure out why something wasn’t working. By the end of the lab, I felt much more confident combining coding with design and structuring my work in a way that actually functions correctly.

Overall, this project showed me how technical skills like coding can directly support communication and storytelling. Being able to present locations visually through an interactive map makes the information much more engaging than just listing it out, and it’s something I could see myself using in future projects.
